# CountDown Tool

The countdown tool will work in those mode，as follows：

1. countdown for specific seconds, and you can pause it until time's up.
2. countdown for a deadline, and you can NOT pause it.
3. stopwatch until you stop it, you can pause it too.
4. pomodoro timer, it will countdown 25 minutes for work and 5minutes for a break then loop it for 4 times.
